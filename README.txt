The program is entirely self contained.
To run the program open the python file and run the code.
The program will force a pip install of the networkx library you do not have to do anything for this to happen.
Use the interactive terminal window to traverse the system.
The system is split into 4 functions (1.1), (1.2), (2.1), (2.2).
If you would like to add your own test data please enter this into the test_file.csv in the data folder in the same format it is currently in.
Test results will be printed to terminal and also written to the test_results.csv file in the data folder.
